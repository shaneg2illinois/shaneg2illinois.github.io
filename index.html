Bigfoot Sightings Visualization Project

The Data | The Analysis (Notebook)

---

Plot 1: Sightings by State

This plot shows Bigfoot sightings aggregated by state using a bar chart. The x-axis encodes state as a nominal variable, while the y-axis encodes the count of sightings as a quantitative measure. I grouped the dataset in Python using a pandas groupby operation to compute total sightings per state before visualizing it in Altair. Color encoding is applied to distinguish states visually, using a categorical color scheme to make comparisons easier across regions.

---

Plot 2: Sightings Over Time

This visualization shows how sightings change over time across different states. The x-axis represents year, and the y-axis represents number of sightings. The color encoding differentiates states, allowing comparison of trends across regions. The dataset was transformed in Python by extracting year from the date column and grouping by year and state. Interactivity is added through selection filtering, allowing users to click on states to highlight or fade other lines, making patterns easier to compare.
